## 1. PROJECT OVERVIEW
This project implements a Natural Language Processing (NLP) pipeline to classify movie reviews into **Positive** or **Negative** sentiments. The research focuses on the "Multitasking Paradox" and linguistic nuances, specifically how **Negation** (e.g., "not good") affects machine learning accuracy.

## 2. TECHNICAL SPECIFICATIONS
- **Algorithm:** Logistic Regression
- **Feature Extraction:** TF-IDF Vectorizer with N-gram range (1, 2)
- **Language:** Python 3.10+
- **Libraries used:** `pandas`, `scikit-learn`, `matplotlib`, `seaborn`

## 3. FILE STRUCTURE
- `main.py`: The primary source code containing the dataset and NLP model.
- `README.md`: This documentation file.
- `confusion_matrix.png`: The visual output of the model's performance.
- `Final_Report.pdf`: The detailed 15-20 page academic report.
